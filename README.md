## 📖 Description

A simple and clean Job Application Tracker that helps you manage your job applications efficiently. You can track which jobs you have applied to, which ones are in interview stage, and which are rejected — all in one place.

---

## 🚀 Project Overview

This project is designed to make job tracking easy and visual. Instead of keeping notes manually, you can quickly update the status of each job with just one click.

Each job card shows key details like company name, role, and status. You can filter jobs based on their status (All, Interview, Rejected), and the system automatically updates counts and UI in real-time.

The interface is responsive and user-friendly, making it suitable for both desktop and mobile users.

## ✨ Features

* 📊 Track total, interview, and rejected jobs
* 🔍 Filter jobs (All / Interview / Rejected)
* 🔄 Real-time UI updates
* ❌ Delete job posts
* 🎯 Status update (Not Applied → Interview / Rejected)
* 📭 "No Data" state handling
* 📱 Responsive design

## 💻 How to Run Locally

1. Download or clone the project

2. Make sure all files are in the same folder:

   * `index.html`
   * `script.js`
   * `img/`

3. Open `index.html` in your browser

That’s it — no build tools or installation needed ✅
